## Getting Help

### In-Game Support

| Command | Description |
|---------|-------------|
| `/help` | View help menu |
| `/faq` | Frequently asked questions |
| `/report <player>` | Report a rule breaker |
| `/ticket` | Open a support ticket |

### Discord Support

For issues that require staff assistance:

1. Go to **#create-ticket** channel
2. Click the appropriate ticket category
3. Describe your issue in detail
4. Wait for a staff response

<Note>
  Average response time is under 24 hours. Please be patient!
</Note>

## FAQ

<Accordion title="How do I join the server?">
  Add `tenebris.umbrixgames.eu` to your Minecraft server list and connect!
</Accordion>

<Accordion title="What version of Minecraft do I need?">
  We recommend the latest version of Minecraft Java Edition. The server supports versions 1.20+.
</Accordion>

<Accordion title="Is the server free to play?">
  Yes! Tenebris is free to play. Optional cosmetic purchases are available to support the server.
</Accordion>

<Accordion title="I lost my items. Can I get them back?">
  Generally, we cannot restore lost items unless there was a confirmed server bug. Always be careful with your valuables!
</Accordion>

<Accordion title="How do I report a bug?">
  Use the **#bug-reports** channel on Discord with details about what happened, when, and steps to reproduce.
</Accordion>

<Accordion title="Can I apply for staff?">
  Staff applications open periodically. Check the announcements channel for opportunities.
</Accordion>
