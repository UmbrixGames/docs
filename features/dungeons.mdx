## How Dungeons Work

<Steps>
  <Step title="Enter the Dungeon">
    Use `/dungeon join <name>` or visit the Dungeon Master NPC
  </Step>
  <Step title="Clear Rooms">
    Defeat all enemies in each room to progress
  </Step>
  <Step title="Defeat the Boss">
    Face the final boss for the best rewards
  </Step>
  <Step title="Claim Rewards">
    Collect loot from the reward chest
  </Step>
</Steps>

## Dungeon Commands

| Command | Description |
|---------|-------------|
| `/dungeon list` | View available dungeons |
| `/dungeon join <name>` | Enter a dungeon |
| `/dungeon leave` | Exit current dungeon |
| `/dungeon party` | Create a dungeon party |

## Dungeon Tiers

Dungeons are organized by difficulty:

### Tier 1 - Beginner
<Info>
  Recommended Level: 1-15
</Info>

- Simple mechanics
- Forgiving boss fights
- Basic crafting materials as rewards

### Tier 2 - Intermediate
<Info>
  Recommended Level: 16-35
</Info>

- Moderate complexity
- Boss phases and mechanics
- Rare equipment drops

### Tier 3 - Advanced
<Info>
  Recommended Level: 36-50
</Info>

- Complex multi-phase bosses
- Requires good gear and coordination
- Epic and custom textured equipment

### Tier 4 - Elite
<Warning>
  Recommended Level: 50+ with optimized gear
</Warning>

- Extreme difficulty
- Party coordination essential
- Legendary drops and exclusive cosmetics

## Party Dungeons

Party dungeons require coordination and class synergy for success.

### Forming a Party

```bash
/party create           # Create a new party
/party invite <player>  # Invite a player
/party accept           # Accept party invitation
/party leave            # Leave current party
```

### Party Roles

For best results, balance your party:

| Role | Responsibility | Best Class |
|------|----------------|------------|
| **Tank** | Hold enemy attention, absorb damage | Warrior |
| **Healer** | Keep party alive | Mage |
| **DPS** | Eliminate enemies quickly | Archer, Warrior |

<Tip>
  Communication is key! Use Discord voice chat for harder dungeons.
</Tip>

## Rewards

Dungeons offer the best rewards in Tenebris:

### Drop Types

- **Equipment:** Weapons, armor, accessories
- **Materials:** Forging components, upgrade stones
- **Currency:** Dungeon tokens for special vendors
- **Cosmetics:** Exclusive skins and effects

### Dungeon Tokens

Earn tokens to spend at the Dungeon Vendor:

| Item | Token Cost |
|------|------------|
| Upgrade Stones | 50 tokens |
| Rare Materials | 100 tokens |
| Epic Equipment | 500 tokens |
| Legendary Chest | 1000 tokens |
