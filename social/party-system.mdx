The Party System allows you to group with other players for cooperative gameplay, especially for tackling dungeons and difficult content.

## Party Overview

Parties provide:

- **Shared dungeon instances** for cooperative play
- **Loot distribution** options
- **Voice/text communication** integration
- **Buff sharing** from party skills

## Party Commands

| Command | Description |
|---------|-------------|
| `/party create` | Create a new party |
| `/party invite <player>` | Invite a player |
| `/party accept` | Accept party invitation |
| `/party deny` | Decline invitation |
| `/party leave` | Leave current party |
| `/party kick <player>` | Remove a player (leader only) |
| `/party list` | View party members |
| `/party chat` | Toggle party chat |
| `/party disband` | Disband the party (leader only) |

## Creating a Party

<Steps>
  <Step title="Create Party">
    Use `/party create` to become the party leader
  </Step>
  <Step title="Invite Members">
    Use `/party invite <playername>` for each player
  </Step>
  <Step title="Configure Settings">
    Set loot rules and other options
  </Step>
  <Step title="Start Your Adventure">
    Enter a dungeon or explore together!
  </Step>
</Steps>

```bash
/party role <player> <tank|healer|dps>
```

<Tip>
  Roles help with automatic targeting and buff priorities!
</Tip>

## Loot Distribution

The party leader can set loot rules:

| Mode | Description |
|------|-------------|
| **Free for All** | First to pick up gets the item |
| **Round Robin** | Items distributed in rotation |
| **Need/Greed** | Vote on who gets rare items |
| **Leader Decides** | Leader assigns all loot |

### Setting Loot Mode

```bash
/party loot <mode>
```

## Party Finder

Looking for a group? Use the party finder:

```bash
/partyfinder browse    # Browse active listings
/partyfinder create    # List your party
/partyfinder join <id> # Request to join a listing
```

### Party Finder Listings

When creating a listing, specify:
- **Content:** Which dungeon or activity
- **Roles needed:** Tank, healer, DPS
- **Level range:** Minimum level required
- **Description:** Any special requirements

| **Mage Blessing** | +5% all stats | 15 blocks |
| **Archer Focus** | +10% crit chance | 15 blocks |
