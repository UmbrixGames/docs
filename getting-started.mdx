Welcome to Tenebris! This guide will help you get started on your adventure.

## Prerequisites

Before you begin, make sure you have:

- Minecraft Java Edition (latest version recommended)
- A valid Minecraft account
- A stable internet connection

## Connecting to the Server

<Steps>
  <Step title="Open Minecraft">
    Launch Minecraft Java Edition and click **Multiplayer**
  </Step>
  <Step title="Add the Server">
    Click **Add Server** and enter the following details:
    - **Server Name:** Tenebris
    - **Server Address:** `tenebris.umbrixgames.eu`
  </Step>
  <Step title="Join and Play">
    Select Tenebris from your server list and click **Join Server**
  </Step>
</Steps>

<Check>
  Once connected, you'll spawn in the Tenebris hub where your adventure begins!
</Check>

## First Steps

When you first join Tenebris, here's what to do:

### 1. Choose Your Class

Tenebris features three unique classes, each with their own playstyle:

| Class | Role | Playstyle |
|-------|------|-----------|
| **Warrior** | Melee DPS / Tank | Close combat, heavy armor, high survivability |
| **Mage** | Ranged DPS / Support | Magic attacks, spells, crowd control |
| **Archer** | Ranged DPS | Precision attacks, mobility, critical hits |

<Tip>
  Choose a class that matches your preferred playstyle. You can experiment with different classes as you progress!
</Tip>

### 2. Start the Tutorial

Follow the in-game tutorial to learn the basics:

- Basic controls and UI
- How to navigate the world
- Introduction to the quest system
- Your first equipment

### 3. Begin Your Island

As a Skyblock server, you'll start with your own island. Use the commands:

```bash
/island create    # Create your island
/island home      # Teleport to your island
/island help      # View all island commands
```

## Essential Commands

Here are the most important commands you'll use:

| Command | Description |
|---------|-------------|
| `/spawn` | Return to the main hub |
| `/island` | Manage your Skyblock island |
| `/quest` | View your active quests |
| `/skills` | Check your skill progress |
| `/party` | Manage your party for dungeons |
| `/guild` | Access guild features |
